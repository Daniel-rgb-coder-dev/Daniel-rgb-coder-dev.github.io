Invoer, Verwerking, Uitvoer en Opslag

Een computer werkt volgens het principe van invoer, verwerking, uitvoer en opslag. Dit betekent dat een computer gegevens ontvangt (invoer), deze gegevens verwerkt, resultaten produceert (uitvoer) en deze resultaten opslaat voor toekomstig gebruik.

Invoer

Invoer is alles wat je in een computer stopt. Dit kan tekst zijn die je typt, een muisklik, een afbeelding die je uploadt, enzovoort.

Verwerking

Verwerking is het daadwerkelijke werk dat de computer doet om de ingevoerde gegevens om te zetten in bruikbare informatie. Dit wordt gedaan door de CPU (Central Processing Unit).

Uitvoer

Uitvoer is het resultaat van de verwerking die door de computer wordt geproduceerd. Dit kan een document zijn, een afdruk, een geluid, enzovoort.

Opslag

Opslag is waar gegevens permanent worden bewaard, zoals op een harde schijf of SSD (Solid State Drive).
